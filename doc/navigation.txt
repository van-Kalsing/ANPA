Класс Targets
	Описание:	коллекция целей (заданий) аппарата, управляет видимостью целей в модели
	Примечания:
		1. Текущая реализация является временной - в дальнейшем планируется получать цели из файла
		
	Метод reset_targets
		Выполняемые функции:	Устанавливает состояние коллекции в начальное состояние,
			отображает все цели в модели
		Вход:	-
		Выход:	-
		
	Метод has_unconfirmed_targets
		Выполняемые функции:	Определяет наличие невыполненных заданий
		Вход:	-
		Выход:	True - невыполненные задания существуют
				False - невыполненные задания не существуют
				
	Метод get_current_target
		Выполняемые функции:	Возвращает текущую цель
		Вход:	-
		Выход:	текущая цель (список координат - [x, y, z])
		
	Метод confirm_current_target
		Выполняемые функции:	Подтверждает выполнение текущего задания, убирает текущую цель из модели
		Вход:	-
		Выход:	-
		
Функция update_ship_engines_forces
	Выполняемые функции:	Управляет силой двигателей аппарата, в зависимости от цели и состояния аппарата
	Вход:	-
	Выход:	-
	Примечания:
		1. Цель считается достигнутой, если текущее расстоние аппарата до цели меньше или равно
			подтверждающему расстоянию (задается параметром объекта Navigation - confirming_distance).
		2. В случае достижения последней цели, если установлен флаг зацикливания
			(задается параметром объекта Navigation - looping_toggle)
			цикл возобновляется, иначе аппарат останвливается.
		3. Выбор силы двигателей происходит при помощи магии
